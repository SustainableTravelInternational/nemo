> A laravel and react based laravel-application to help save the reefs

### 🏠 [Homepage](https://supportnemo.com/)

## Prerequisites

- npm >=6.13.0
- node >=11.15.0
- docker >=19.03.0

## Install

```sh
npm install
```

## Usage

```sh
npm run start
```
